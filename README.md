# CGPA Calculator Analytics (Case Study)

## Overview
This project started as a simple CGPA calculator built to assist students during exam/result periods.

Over time, it evolved into a widely used utility, gaining significant traction through organic search traffic and serving real users.

---

## Impact & Performance

- 7.7K+ clicks from Google Search  
- ~23K impressions  
- ~32% Click-Through Rate (CTR)  
- Ranked on the first page of Google  

The application experienced recurring traffic spikes during academic result cycles, indicating strong real-world demand.

---

## Key Insight

Solving the right problem at the right time matters more than building complex systems.

---

## Lifecycle

- Built as a utility tool during academic result periods  
- Gained traction organically via Google Search  
- Usage peaked during exam/result cycles  
- Archived as the R20 regulation cycle concluded with the final batch
